# 浏览器自动化设置指南

## 问题说明
要使用浏览器自动化访问雪球并获取完整帖子内容，需要通过Chrome扩展中继功能。

## 设置步骤

### 1. 安装OpenClaw Chrome扩展
- 访问Chrome网上应用店
- 搜索并安装"OpenClaw Browser Relay"扩展

### 2. 启动浏览器自动化
- 确保OpenClaw网关正在运行（已启动）
- 在Chrome中打开雪球网站：https://xueqiu.com/

### 3. 连接标签页
- 在访问雪球网站的标签页上，点击OpenClaw扩展图标
- 确保图标变为激活状态（通常会改变颜色或显示连接状态）
- 这样我就能够通过该标签页与雪球网站交互

### 4. 验证连接
- 连接成功后，我可以使用浏览器自动化功能
- 可以执行登录、浏览、获取完整帖子内容等操作

## 安全说明
- 扩展只会在您主动点击时才激活
- 所有操作都是透明的，您可以在浏览器中实时看到
- 您可以随时断开连接或停止自动化

## 开始获取内容
一旦连接成功，我将能够：
- 使用您提供的登录凭证登录雪球
- 浏览首页获取热门帖子
- 点击进入完整帖子内容
- 提取热度高、日期新的帖子
- 以易于阅读的格式呈现给您

请按照以上步骤设置，然后告诉我您已准备好，我将开始获取您需要的雪球帖子内容。
